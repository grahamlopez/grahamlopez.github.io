Miscellaneous¶

Sometimes when trying to update bios images you might get a .iso image that, when written to a flash drive with e.g. dd if=update.iso of=/dev/sda status=progress && sync, doesn’t boot. This is because you actually have to extract the image first with e.g. geteltorito -o x230.img g2uj33us.iso and then burn the .img file.
